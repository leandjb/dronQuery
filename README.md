# DronQuery

Aplicación web interactiva para consultar el espacio aéreo restringido para operaciones con drones (RPAS/UAS) en Colombia.

## Descripción

DronQuery es una herramienta basada en [Leaflet](https://leafletjs.com/) que proporciona una visualización clara y accesible de las zonas restringidas para operaciones de vehículos aéreos no tripulados (RPAS - Remotely Piloted Aircraft Systems / UAS - Unmanned Aircraft Systems) en territorio colombiano.

## Características

- Consulta interactiva del espacio aéreo restringido
- Mapa interactivo basado en Leaflet
- Información geoespacial de Colombia
- Interfaz intuitiva y responsiva

## Cómo usar

1. Abre `index.html` en tu navegador
2. Explora el mapa interactivo
3. Consulta las zonas restringidas para operaciones con drones

## Tecnologías utilizadas

- **Leaflet**: Biblioteca de mapas interactivos
- **HTML/CSS/JavaScript**: Tecnologías web estándar
- **GeoJSON**: Formato para datos geoespaciales
